# DNA部分关于图片的解释：
##### 刚扫过这个文件，发现我的图片我没有太多的文字描述，现在花点时间补下，
##### 德塔 类人的dna由 父亲和母亲的2条遗传肽链进行配对遗传，父亲和母亲的肽链皆由 2根链组成，1根是微分的功能函数组合成某功能应用list
##### 另一根是功能的掩码，主要用来做备份修复等功能。 
##### 关于微分的功能函数组合 是指 将某一种应用功能进行函数的不断细化和微分，成为aopm 和vpcs的 initon 初始量子列阵函数集合，不同的应用功能
##### 由不同的这些列阵initon组合，每一个initon会有一个mask的标注， 比如吃饭划分细节就是 举手 托碗 拿起筷子 夹菜 送入口中，
##### 喝水 则是  举手 托杯 送入口中， 他们相同的initon部分 比如举手 和送入口中的mask 码为 1， 这些0和1 组成一些功能片段然后组成应用片段，
##### 形成dna记忆函数，  aopm 和 vpcs 我之后的工程会进行深入研究。
